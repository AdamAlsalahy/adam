## التثبيت والتشغيل

### المتطلبات
- Python 3.8+
- pip
- virtualenv (اختياري لكن موصى به)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
cd /path/to/project
```

2. **إنشاء بيئة افتراضية (اختياري)**
```bash
python -m venv venv
source venv/bin/activate  # على Linux/Mac
venv\Scripts\activate     # على Windows
```

3. **تثبيت المكتبات**
```bash
pip install -r requirements.txt
```

4. **إعداد متغيرات البيئة**
```bash
cp .env.example .env
# قم بتعديل ملف .env بإضافة بيانات الإنتاج
```

5. **تطبيق الهجرات (Migrations)**
```bash
python manage.py migrate
```

6. **إنشاء حساب المسؤول**
```bash
python manage.py createsuperuser
```

7. **جمع الملفات الثابتة (للإنتاج)**
```bash
python manage.py collectstatic --noinput
```

8. **تشغيل الخادم**
```bash
# للتطوير
python manage.py runserver

# للإنتاج (استخدام Gunicorn)
gunicorn core.wsgi:application --bind 0.0.0.0:8000
```

## الوصول إلى لوحة التحكم

بعد التثبيت، يمكنك الوصول إلى لوحة التحكم من:
```
http://localhost:8000/admin/
```

استخدم بيانات المسؤول التي أنشأتها في الخطوة 6.

## تحسينات الأمان

### للإنتاج (Production)
1. تغيير `DEBUG` إلى `False` في `.env`
2. تغيير `SECRET_KEY` إلى مفتاح عشوائي قوي
3. تعيين `ALLOWED_HOSTS` بأسماء النطاقات الفعلية
4. تفعيل HTTPS و `SECURE_SSL_REDIRECT = True`
5. تفعيل `SESSION_COOKIE_SECURE = True`
6. تفعيل `CSRF_COOKIE_SECURE = True`
7. استخدام قاعدة بيانات قوية (PostgreSQL بدلاً من SQLite)
